# ZamasuSearch
Zamasu Search is a Python GUI Based for Searching Exploit from exploit-database.com

## Install
```
git clone https://github.com/aancw/ZamasuSearch.git
cd ZamasuSearch
python ZamasuSearch.py
```
## Dependency

* gitpython
* python-qt5

## Requirements

* Python 2.7 or newer
* Git 1.7.0 or newer
